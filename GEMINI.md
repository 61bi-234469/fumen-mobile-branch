# fumen-for-mobile (tetfu-sp) プロジェクト分析概要

このプロジェクトは、モバイルおよびデスクトップ環境で動作する、**テトリスの譜面（fumen）の閲覧および編集を行うPWA（Progressive Web App）**です。軽量な仮想DOMライブラリとCanvas描画を組み合わせ、スムーズなテトリスの盤面操作を実現しています。

## 主要な技術スタック

*   **フレームワーク**: [Hyperapp 1.x](https://github.com/jorgebucaran/hyperapp)（軽量な状態管理・仮想DOMライブラリ）
*   **言語**: TypeScript（`strict` モード有効）
*   **描画エンジン**: Konva（HTML5 Canvasを利用して10x23のテトリスフィールドを描画）
*   **ビルドツール**: Webpack 5
*   **テスト**: Jest（単体テスト） / Cypress（E2Eテスト・UIテスト）
*   **国際化**: i18next（英語と日本語をサポート）

## アプリケーションのアーキテクチャ

Hyperappによる**「ユーザー入力 → アクション → 状態更新 → ビュー再レンダリング」**という一方向データフロー（Immutable）を採用しています。

1.  **アクション (`src/actions/`)**:
    ドメインごとに分割された純粋関数群です。
    *   `animation.ts` (再生/一時停止などのコントロール)
    *   `field_editor.ts` (ブロック操作)
    *   `memento.ts` (Undo/Redoの履歴管理)
    *   `pages.ts` (ページナビゲーション)
2.  **ビューとモード (`src/views/`)**:
    アプリケーションは大きく2つの画面モードを持っています。
    *   **Reader (`reader.ts`)**: 譜面の閲覧・アニメーション再生モード
    *   **Editor (`editor/`)**: 譜面作成モード（ブロックの描画、テトロミノピースの配置、塗りつぶし、またフラグ設定など複数のサブ編集モードを持ちます）
3.  **Fumen エンコード/デコードコア (`src/lib/fumen/`)**:
    v115 および v110 フォーマットの譜面コードのエンコード・デコード処理を担う中核ライブラリが内包されています。クイズコメント（`#Q=[]()`）のパース機能も備わっています。

## 開発・運用に関する特徴

*   **PWA対応**: Workboxを使用し、オフラインやモバイル端末でのネイティブライクなアプリインストールをサポートしています。
*   **コーディング規約**: TSLint (Airbnb config) を導入し、インデントは4スペース、1行最大約120文字の制約が設けられています。
*   **開発コマンド**:
    *   `yarn webpack` / `yarn webpack-prod`: 開発・本番ビルド
    *   `yarn serve`: ローカルサーバー起動
    *   `yarn test`: Jestによるロジックのテスト
    *   `yarn cy-open` / `yarn cy-run`: CypressによるUI・E2Eテスト
